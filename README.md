# 📊 Telco Customer Churn: Multivariate Exploratory Data Analysis  

## 🚀 Project Overview  
This project performs an **end-to-end Exploratory Data Analysis (EDA)** on a dataset of **7,000+ customers** to identify critical behavioral and demographic risk factors contributing to customer attrition.  

By leveraging statistical visualization techniques, the analysis quantifies key churn drivers and supports **data-driven retention strategies** in the telecommunications sector.  

---

## 🎯 Objective  
- Identify key drivers of customer churn  
- Analyze behavioral and demographic patterns  
- Quantify attrition risk  
- Enable data-driven retention strategies  

---

## 📊 Key Technical Insights  
The analysis revealed several statistically significant drivers of churn:  

- **Contractual Friction:**  
  Customers on **Month-to-Month contracts** exhibit high churn (~42.7%), whereas **Two-Year contracts** show strong customer retention (~2.8% churn).  

- **Payment Method Impact:**  
  **Electronic Check users** contribute ~45% of total churn, indicating potential friction in the payment experience.  

- **Early-Tenure Risk:**  
  Churn is highest within the first year (~50%), making onboarding the most critical phase.  

- **Demographic Factors:**  
  **Senior Citizens** show higher churn rates, suggesting a need for targeted retention strategies.  

---

## 🔍 Analysis Workflow  
1. **Data Preprocessing**  
   - Handled missing values in `TotalCharges`  
   - Corrected data types  
   - Transformed categorical variables for better interpretability  

2. **Univariate Analysis**  
   - Analyzed churn distribution  
   - Established baseline churn rate (~26.54%)  

3. **Bivariate Analysis**  
   - Explored relationships between churn and:  
     - Gender  
     - Tenure  
     - Contract Types  
     - Internet Services  

4. **Service Impact Study**  
   - Evaluated the impact of services like Tech Support and Online Security on retention
